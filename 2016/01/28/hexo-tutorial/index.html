hexo tutorial

2016-01-28 |

安装hexo:

npm install hexo

初始化hexo:
hexo init

开始安装主题：参考maupassant的github

I.对于hexo3.0以后的版本，hexo-deployer-git被独立出来了需要单独进行安装.

npm install hexo-deployer-git

II.配置好后：

hexo d

结果报：

bash: /dev/tty: No such device or address
error: failed to execute prompt script (exit code 1)
fatal: could not read Username for 'https://github.com': No error
FATAL Something's wrong. Maybe you can find the solution 
here: http://hexo.io/docs/troubleshooting.html

后将hexo目录下的_config.yml文件reploy项改为如下后正常：

deploy:
    type: git
    repository: ssh://git@github.com/yourname/yourname.github.io
    branch: master
